# Smart Hostel CMS

A Java Swing + MySQL based Complaint Management System.

---

## 🔹 Features
- Student Login
- Warden Login
- Complaint Registration
- Complaint Status Tracking (Pending / Resolved)
- Warden Dashboard to manage complaints

---

## 🔹 Technologies Used
- Java (Swing)
- MySQL
- JDBC

---

## 🔹 Database
Database Name: hostel_system

Tables:
- users
- students
- complaints

## 🔹 How to Run
1. Import project in VS Code
2. Add MySQL connector JAR
3. Update DBConnection.java password
4. Run Login.java
5. Note:
The MySQL Connector JAR is not included in the repository to keep the project lightweight. It can be easily downloaded from the official MySQL website.
